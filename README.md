# jquery.smoothscroll.js

jQuery plugin to animate scrolling to anchor links

## Usage

```
$('a[href*="#"]').smoothscroll();
```

## Options and Settings


```
$('a[href*="#"]').smoothscroll({
  speed: 500,
  easing: 'swing',
  hash: true,
});
```
